Fortran-C binding for computing the Tangent Stiffnes Matrix and Residuum. All Values must be passed by reference!

Parameters
[in]vWorking vector for AceGen, size fixed to 1060
[in]d[Input] Material Parameters size 2
[in]ul[Input] Solution vector. Size ul(3,10) in Fortran(column major). Must be sent in as a vector in column major.
[in]ul0Currently unused, send a vector of size 30.
[in]xl[Input] Position of nodes in reference coordinates. Size xl(3,10) in Fortran(column major). Must be sent in as a vector in column major form.
[in]s[Output] Tangent Stiffness Matrix. Size s(30,30) in Fortran(column major). Must be sent in as a vector in column major form.
[in]p[Output] Residuum vector, Size p(30).
[in]htHistory parameters at time n. Currently unused.
[in]hpHistory parameters at time n+1. Currently unused.
