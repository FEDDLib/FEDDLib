\begin{DoxyTemplParams}{Template Parameters}

{\em LO} & The local ordinal type. The is the index type for local indices \\

This class provides a function for constructing objects of classes derived from \mbox{\hyperlink{classFEDD_1_1AssembleFE}{Assemble\+FE}}. In particular, an \mbox{\hyperlink{classFEDD_1_1AssembleFE}{Assemble\+FE}} object is constructed as follows\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{AssembleFE assmeblyFe = \mbox{\hyperlink{classFEDD_1_1AssembleFEFactory_ac895d65acf2626100832586df84d6a9c}{AssembleFEFactory<>::build}}(\textcolor{stringliteral}{"{}problemType"{}},flag,nodesRefConfig,params);}

\end{DoxyCode}
 If a new element assembly is implemented, it has to be added to the \mbox{\hyperlink{classFEDD_1_1AssembleFEFactory_ac895d65acf2626100832586df84d6a9c}{build()}} function. Only then, this element assembly can be executed within the FEDDLib.
